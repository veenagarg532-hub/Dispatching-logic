Features Implemented
====================

✅ Discrete-Event Simulation (SimPy)
   - Lot arrivals (Poisson/deterministic)
   - Toolset queue management
   - Product-dependent processing times
   - Tool-product cross-qualification
   - Metric collection at intervals

✅ Scheduling Engine (4 Strategies)
   - FIFO (First-In-First-Out)
   - Least-Loaded-Tool
   - Random-Qualified-Tool
   - Optimised (Shortest Processing Time)

✅ Web Interface
   - Responsive single-page app
   - Interactive configuration panel
   - 4 synchronized metric plots
   - Summary statistics
   - Pedagogical concept explainer
   - Run/Reset controls

✅ Backend API
   - GET /api/scenario (metadata)
   - POST /api/simulate (run simulation)
   - GET /health (health check)
   - Interactive docs at /docs

✅ Documentation
   - Comprehensive README
   - Detailed architecture docs
   - Quick start guide
   - Installation instructions
   - Project summary
   - Delivery checklist
